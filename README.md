# Floppy Disk

A retro Flappy Bird-style arcade game where you guide a floppy disk through 7 eras of computing history, battling storage bosses from the Hard Disk era all the way to the Cloud.

**Play in your browser:** https://roelofdupreezrjd-maker.github.io/floppy-disk/

## Game modes

- **Original** — classic flappy mechanics
- **Boss Fight** — fight a boss every 10 pipes
- **Kiddies** — drag-to-move mode for younger players

## Speed levels

- **Beginner** — floaty and forgiving
- **Middle** — balanced
- **Pro** — original Flappy Bird feel

## Bosses

Hard Disk → Zip Drive → CD/DVD → USB Flash → SSD → NVMe → Cloud Storage → (secret final boss)
